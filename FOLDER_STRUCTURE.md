## Step-by-Step Setup Guide

### Step 1 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Download datasets
- Rice: https://data.mendeley.com/datasets/hx6f852hw4/2
- Cotton: https://data.mendeley.com/datasets/b3jy2p6k8w/2

Place them under:
```
disease_detection/Rice_Cotton_disease_Data/Rice_Disease_Data/Rice/
disease_detection/Rice_Cotton_disease_Data/Cotton_Disease_Data/Cotton/
```

### Step 3 — Generate augmented data (if not already done)
```bash
cd disease_detection
python augmenter_offline.py
```

### Step 4 — Train disease models
```bash
# Train all backbones, both crops (recommended)
python train_disease.py

# Or train only one crop with one backbone for a quick test
python train_disease.py --crop rice --backbone EfficientNetV2B0
```

### Step 5 — Update Flask files
1. Copy `flask_integration/services.py` → `disease_detection/services.py`
2. Copy `flask_integration/config.py`   → `config.py`
3. Copy `flask_integration/requirements.txt` → `requirements.txt`

### Step 6 — Run the app
```bash
python app.py
```

### Step 7 — Test inference
```bash
# CLI test
python scripts/infer_disease.py --image path/to/test.jpg --pretty

# Evaluate on test set
python scripts/evaluate_disease.py --crop rice
```

## API Response Format

**POST /api/disease/detect**
```json
{
  "crop":       "Rice",
  "disease":    "Leaf Blast",
  "confidence": "97.34%",
  "top3": [
    {"class": "Leaf Blast",      "confidence": "97.3%"},
    {"class": "Brown Spot",      "confidence": "1.9%"},
    {"class": "Sheath Blight",   "confidence": "0.5%"}
  ],
  "advice": "Expert AI advice from Mistral LLM...",
  "note":   "Image quality enhanced (blur score: 45.2)"
}
```

**Error response (low confidence):**
```json
{
  "crop":       "Cotton",
  "disease":    "Leaf Variegation",
  "confidence": "38.10%",
  "warning":    "Low confidence (38.1%). Please upload a clearer, closer image of the leaf.",
  "advice":     "..."
}
```
